## A customizable password generator built using React Hooks.

## Features

- Generate secure passwords
- Copy to clipboard
- Include numbers
- Include special characters
- Adjustable password length

---

## Tech Stack

- React
- Tailwind CSS
- Vite

---

## React Hooks Used

- useState
- useEffect
- useCallback
- useRef
