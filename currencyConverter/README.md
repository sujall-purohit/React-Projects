# Currency Converter

## A real-time currency converter built using React and custom hooks.

## Features

- Real-time conversion
- Currency swap feature
- Reusable components
- Multiple currencies

---

## Tech Stack

- React
- Tailwind CSS
- Custom Hooks
- Vite
