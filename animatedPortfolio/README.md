# Animated Portfolio

A modern and responsive portfolio website built using React.js, Tailwind CSS, and GSAP animations.

## Features

- Responsive Design
- Smooth Animations
- Modern UI
- Scroll Navigation
- Reusable Components
- Contact Form

## Tech Stack

- React.js
- Tailwind CSS
- GSAP
- Vite

## Installation

```bash
npm install
npm run dev
```
