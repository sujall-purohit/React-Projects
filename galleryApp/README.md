# Image Gallery App
A responsive image gallery application using Picsum API.
---
## Features
- Fetch images from API
- Pagination support
- Loading state handling
- Modern responsive gallery
6
---
## Tech Stack
- React
- Axios
- Tailwind CSS
- Vite
---
## API Used
https://picsum.photos/
---
## Installation
```bash
npm install
```
---
## Run Project
```bash
npm run dev
```
