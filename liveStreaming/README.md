# 🔴 Live Streaming & Video Meeting App

A modern real-time video conferencing and live streaming application built using React and ZEGOCLOUD UIKit.

---

## ✨ Features

- 🎥 Create and join live meeting rooms
- 🔗 Dynamic room sharing
- 👥 Real-time video conferencing
- ⚡ Instant room joining
- 📱 Fully responsive UI
- 🎨 Modern glassmorphism design
- 🌐 React Router based navigation

---

## 🛠 Tech Stack

- React
- React Router DOM
- Tailwind CSS
- ZEGOCLOUD UIKit
- Vite
