# Video Meeting App

A modern real-time video conferencing application built using React and
ZEGOCLOUD.

---

## Features

- Create and join meeting rooms
- Real-time video conferencing
- Dynamic room routing
- Responsive UI
- Beautiful background interface

---

## Tech Stack

- React
- React Router DOM
- Tailwind CSS
- ZEGOCLOUD UIKit
- Vite

---

## Installation

```bash
npm install
```

---

## Run Locally

```bash
npm run dev
```
