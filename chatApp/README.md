# Real-Time Chat Application

A modern real-time chat application built using React, Firebase, Zustand, and
Cloudinary.

---

## Features

- Authentication System
- Real-time messaging
- User profile editing
- Image sharing
- Block & unblock users
- Search chats
- Cloudinary image uploads
- Toast notifications
- Responsive modern UI

---

## Tech Stack

- React
- Firebase Authentication
- Firestore Database
- Zustand
- Cloudinary
- React Router DOM
- Tailwind CSS
- React Toastify
- Vite

## Installation

```bash
npm install
```

---

## Environment Variables

Create a `.env` file:

```env
VITE_API_KEY=
VITE_AUTH_DOMAIN=
VITE_PROJECT_ID=
VITE_STORAGE_BUCKET=
VITE_MESSAGING_SENDER_ID=
VITE_APP_ID=
VITE_CLOUDINARY_CLOUD_NAME=
VITE_CLOUDINARY_UPLOAD_PRESET=
```
