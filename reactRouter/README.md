# React Router Practice Project

A React project created to practice routing concepts including nested routes,
loaders, layouts, and dynamic routing.

---

## Features

- Nested Routing
- Dynamic Routes
- Shared Layouts
- Navbar Navigation
- GitHub API Integration
- Responsive Design

---

## Tech Stack

- React
- React Router DOM
- Tailwind CSS
- Vite
